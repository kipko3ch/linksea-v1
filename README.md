# LinkSea

A modern link management platform built with Next.js and Supabase.

## Features

- 🔗 One link for all your content
- 📊 Real-time analytics
- 🎨 Customizable profiles
- 📱 Mobile-optimized
- 🔒 Secure authentication
- 🌙 Dark mode support

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Framer Motion
